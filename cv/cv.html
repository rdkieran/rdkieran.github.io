Projects

Audio Sequencer

Python, Whisper (OpenAI)

> Audio Sequencer on GitHub

Developed with Sandside Studios to help in sequencing two-person dialogue recordings where each speaker for a given dialogue script is recorded separately.

Utilising AI-generated word-level transcriptions of given audio files and the original dialogue script, the python script locates spoken dialogue in each transcript, finds matches within the original dialogue script, extracts matched audio using transcript timestamps, and organises the files into the order of the script.

The individual audio files can then be spliced together to produce one continuous audio file consistent with the dialogue script.
